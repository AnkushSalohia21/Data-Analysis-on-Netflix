# Data-Analysis-on-Netflix
This Netflix Data Analysis project is created to analyze user subscription data and revenue trends using Python. The project includes data cleaning, analysis, and interactive visualizations to understand patterns related to age, subscription type, content duration, and monthly revenue. Tools like Pandas, Plotly, and Streamlit are used to build an interactive dashboard. This project is developed by Ankush Salohia as a learning and portfolio-based data analysis project.
Steps to run Streamlit 
!. open file "streamlitnf.py"
2. click on run button
3. type "streamlit run streamlitnf.py" in terminal 
4. you will se all the results in local host
